Projenin temel özellikleri ve JavaScript fonksiyonları bu modülde yönetilmektedir.

Bu klasör, uygulamanın ana işlevlerini, API çağrılarını ve dinamik yapısını yöneten temel kaynak kodlarını (örn: `app.js`) barındırır.

**Mimari Notu:** Vercel, Netlify gibi bulut platformlarının projeyi sorunsuz derleyip yayına alabilmesi için web mimarisi standartlarına uyulmuştur. Bu nedenle uygulamanın ana giriş noktası (`index.html`), global tasarım dosyası (`styles.css`) ve yapılandırma dosyaları (`package.json`) bilinçli olarak ana dizinde (root) bırakılmıştır.
